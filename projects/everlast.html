Everlast: Undying Tale

Next.js TypeScript React Tailwind CSS live

I built the marketing site for Everlast: Undying Tale, an open-world action RPG by InSpades Studios.

  • A marketing site with a news blog, a careers page, and landing pages the team can spin up themselves, built in Next.js and Tailwind.
  • A content workflow so non-technical editors can write, edit, and publish without touching the code.
  • Access controls so only authorised team members can publish changes.
  • A section-based page builder, so editors assemble pages from reusable components instead of building from scratch.

It's live at inspades.net, and the game's on Steam.
